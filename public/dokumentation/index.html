Registrierung

Um die Platform nutzen zu können brauchst man einen Invite-link. Es gibt zwei Arten von Invite-Links:

  • Klassen Invite-link
  • Lehrer Invite Link

Klassen Invite-Links können mehrfach benutzt werden. Es wird pro Klasse ein Klassen Invite-Link erstellt mit dem sich die Schüler registrieren können.
Lehrer Invite-Links können nur einmalig benutzt werden.

Um als Lehrer einen Invite-Link zu bekommen, schick einfach eine Mail an zgk@mxis.ch.
Schüler können sich nur über den Klassen Invite-Link registrieren. Wenn deine Klasse noch nicht registriert ist, schick einfach eine Mail an zgk@mxis.ch.

Der Invite Link weißt einen Automatisch einer Klasse zu. Es muss nur der Name, Email Adresse und ein Passwort angegeben werden. Die Email Adresse und das Passwort werden für die Anmeldung benötigt, der Name wird anderen innerhalb der Klasse angezeigt.

Anmeldung

Man kann sich jederzeit auf unterschiedlichen Geräten anmelden. Gehe dazu einfach auf https://zgk.mxis.ch/login oder klicke den “Login” button im Navigationsmenü.

Benutzung

Aufgaben Übersicht

Wenn man die Website besucht, landet man zunächst auf der Aufgaben Seite. Hier sieht man alle Aufgaben seiner Klasse. Jede Aufgabe ist in ihrer eigenen Zeile. Es wird das Fach, das Abgabedatum, der Aufgaben Text, der Dateiname und der Download button angezeigt. Über diesen kann man die Aufgaben Datei herunterladen.

Lösugen hochladen/anschauen

Schüler:

Um eine Lösung hochzuladen klicke einfach in der Übersicht auf die jeweilige Aufgabe. Es öffnet sich eine neue Seite bei der man seine Lösung hochladen kann.

Tipp: Du kannst eine oder mehrere Dateien als Lösung hochladen

Lösungen können nicht wieder gelöscht werden. Es kann jedoch eine Änderung über den gleichen Weg hochgeladen werden (der Lehrer sieht dann beides).

Lehrer:

Um die abgebenen Lösungen herunterzuladen klicke einfach in der Übersicht auf die jeweilige Aufgabe. Es öffnet sich eine neue Seite auf der nochmals die Aufgabe sowie eine Tabelle mit allen Schülern der Klasse angezeigt wird. In der Tabelle sieht man sofort wer noch keine Lösung abegeben hat. Wenn der Haken neben einem Schüler grün ist kann über das blaue Download Icon daneben die Lösung heruntergeladen werden.

Tipp: Die Tabelle kann auch als PDF exportiert werden. Klicke dazu einfach auf den Button am Ende der Tabelle. Mit dem Button daneben können auch alle Lösungen als ZIP heruntergeladen werden.

Schüler können sowohl mehrere Dateien als eine Lösung als auch eine zweite Lösung (z.B eine Verbesserung) hochladen. Alle Dateien werden in einer ZIP Datei heruntergeladen. Das Abgabedatum ist jeweils der Zeitpunkt als die erste Lösung hochgeladen wurde.

Eine neue Aufgabe erstellen (nur Lehrer)

Um eine neue Aufgabe zu erstellen klicke einfach auf den “Neu” button im Navigationsmenü oder gehe auf diesen Link: https://zgk.mxis.ch/new.
Hier muss das Fach, die Klasse, ein Aufgabentext und ein Abgabetermin angegeben werden. Der Dateiupload ist optional. Um die Aufgabe zu erstellen klicke einfach auf den “Erstellen” button.

Account verwalten

Passwort ändern

Dein Passwort kann auf der Account Seite geändert werden. Klicke dazu entweder auf “Account” im Navigationsmenü oder gehe auf diesen Link: https://zgk.mxis.ch/account.

Telegram/Discord Bot hinzufügen

Um den Telegram/Discord Bot hinzuzfügen brauchst du einen Bot Key. Diesen findest du auf der Account Seite. Klicke dazu entweder auf “Account” im Navigationsmenü oder gehe auf diesen Link: https://zgk.mxis.ch/account.
